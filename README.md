# Journal App

A personal productivity and reflection platform built using Python, SQLite, HTML, CSS, and JavaScript.

## Features

- Daily journal entries
- Mood tracking
- Water intake monitoring
- Voice note recording
- Structured data storage using SQLite

## Tech Stack

- Python
- SQLite
- HTML
- CSS
- JavaScript

## Purpose

The project aims to combine journaling, habit tracking, and self-reflection into a single platform.
